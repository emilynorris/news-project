Get an article

Use this API call to retrieve an individual article filtered by the article_id. The request will return details of the article. This is filtered using the article_id.

Example request

GET /api/articles/5

Example response

      article: {
        article_id: 5,
        title: 'UNCOVERED: catspiracy to bring down democracy',
        topic: 'cats',
        author: 'rogersop',

        votes: 0,
        article_img_url: 'https://images.pexels.com/photos/158651/news-newsletter-newspaper-information-158651.jpeg?w=700&h=700'
      }
    }

Update votes for an article

Use this API call to update the votes for an article and return the updated article. This should be used if you want to increase or decrease the votes for a specific article. This is filtered using the article_id.

Example request

PATCH /api/articles/1
Request body
            {
                "inc_votes": 37
            }
Example response 
                   {
      updatedArticle: {
        article_id: 1,
        title: 'Living in the shadow of a great man',
        topic: 'mitch',
        author: 'butter_bridge',
        body: 'I find this existence challenging',
        created_at: '2020-07-09T20:11:00.000Z',
        votes: 137,
        article_img_url: 'https://images.pexels.com/photos/158651/news-newsletter-newspaper-information-158651.jpeg?w=700&h=700'
      }
    }
